# Simon Game Clone

A modern, responsive digital recreation of the classic 1978 electronic memory game. This project challenges players to memorize and replicate increasingly complex sequences of flashing lights and audio tones.

## 🔗 Live Demo
Play the live, interactive game directly in your browser here:  
👉 **[Launch Live Game](https://simonsgame01.netlify.app/)**

## 🚀 Features
* **Classic Gameplay**: Replicates the authentic random sequence building mechanism of the original device.
* **High Score Tracking**: Automatically saves and displays the player's highest score using local storage.
* **Responsive Design**: Optimised for fluid gameplay across mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
* **Audio**: Web Audio API / Native Audio elements

## 📦 Getting Started

Follow these steps to run the game locally on your machine.

### Prerequisites
You only need a modern web browser (Google Chrome, Mozilla Firefox, Safari, or Microsoft Edge) installed.

### Installation & Running
```bash
# 1. Clone the repository
git clone https://github.com

# 2. Navigate into the project folder
cd simon-game

# 3. Open the game in your browser
# On macOS:
open index.html

# On Windows:
start index.html
```

## 🎮 How to Play
1. Click the **Start** button or press any key to initiate the first round.
2. Observe the initial color flash and listen to its corresponding musical note.
3. Click the identical color pad to match the sequence.
4. Watch closely as the game repeats the original pattern and adds one new random color to the end of the chain.
5. Continue repeating the expanding sequence. A single incorrect press triggers the failure buzzer and ends the game.
